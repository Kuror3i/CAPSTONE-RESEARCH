Use this prompt:

---

**Design a simple, modern, and professional customer-facing website for a co-working space / study hub workspace management system.**

The website is for external users who want to:

* view the co-working space
* check available workspaces
* reserve a workspace online
* apply for membership
* see the space location
* use an interactive workspace map as the main selection tool

Keep the design **simple**, **clean**, and **easy to understand**.
Do not make it look too complex or too much like a dashboard.

## Main design direction

* web-based interface
* modern and professional
* clean white or soft light background
* blue or deep teal accent color
* rounded cards
* subtle shadows
* strong spacing
* highly readable typography
* responsive for desktop, tablet, and mobile
* premium but minimal feel

---

# MAIN GOAL OF THE WEBSITE

The website should help users:

1. understand the space
2. view available workspaces
3. explore the layout through the interactive map
4. reserve online
5. apply for membership

The **interactive workspace map** should be one of the most important parts of the website.

---

# REQUIRED PAGES / SECTIONS

## 1. Home Page

Create a clean homepage with:

### Hero section

* strong headline about finding and reserving the right workspace
* short supporting text
* primary CTA: **Reserve a Workspace**
* secondary CTA: **View Workspace Map**

### Quick overview section

Short value cards for:

* real-time workspace visibility
* online reservation
* QR-based access
* easy space selection

### Workspace preview section

* show a few workspace types such as:

  * open desk
  * private desk
  * meeting room
  * study area
* clean cards with short descriptions

### Location section

* include a Google Map embed area
* show address, operating hours, and contact info
* include button: **Get Directions**

### Membership section

* short explanation of membership
* CTA: **Apply for Membership**

---

## 2. Workspace Discovery / Reservation Page

This is the most important page.

### Main layout

* simple page with the **interactive workspace map**
* map should be the central focus
* include filters above or beside the map:

  * workspace type
  * date
  * time
  * duration
  * available only

### Workspace map behavior

* users can click on a desk or room directly on the map
* map should show workspace status using colors:

  * available
  * reserved
  * occupied
  * unavailable

### Very important requirement

When a user selects a workspace on the map, show a **workspace detail panel/card** similar in idea to Airbnb:

#### Show:

* **one attached photo** of the selected workspace
* workspace name
* workspace type
* short description
* availability status
* price / rate
* button: **Reserve This Workspace**

The photo should help the user better understand what that selected space looks like, but the map should still remain the main selection tool.

### Style of the detail panel

* modern side panel or modal
* clean and premium
* one featured photo only
* simple layout
* not overloaded with too much text

---

## 3. Reservation Flow

The user should be able to reserve after selecting a workspace.

### Keep it simple and step-based

Use a multi-step reservation flow:

#### Step 1

Choose date, time, duration

#### Step 2

Select preferred workspace from the map

#### Step 3

Select alternative workspaces if needed

#### Step 4

Enter details:

* full name
* email
* optional contact number
* privacy consent checkbox

#### Step 5

Payment summary and payment method

#### Step 6

Reservation confirmation with QR/reference notice

### Important

The flow should feel simple, clear, and modern.
Do not overcomplicate it.

---

## 4. Membership Application Page

Simple page for users who want to apply as a workspace member.

### Include

* short explanation of membership
* benefits of membership
* clean form with:

  * full name
  * email
  * contact number
  * user type
  * preferred usage frequency
  * consent checkbox
* button: **Submit Membership Application**

---

## 5. Member Login Entry

Create a simple login entry page for members.

### Include

* welcome text
* email
* password
* sign in button
* forgot password
* link to membership application

---

# UI / UX RULES

## 1. Keep it simple

This website should not feel heavy or complicated.

## 2. The map is the core

The interactive workspace map should be the centerpiece of the reservation experience.

## 3. Workspace photo appears on selection

When a workspace is clicked on the map, show:

* one featured image
* short workspace details
* reserve CTA

This should feel similar to Airbnb in principle:

* select a place
* see its image and details
* decide if you want it

## 4. Photos should support, not dominate

The website can have photos, but the booking decision should still rely mainly on:

* map visibility
* space status
* workspace details

## 5. Responsive design

* desktop: map with side detail panel
* tablet: stacked map and detail card
* mobile: map on top, bottom sheet or card for selected workspace details
